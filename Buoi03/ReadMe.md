# Hướng dẫn ngôn ngữ C# - Buổi 03

## 1. Câu lệnh điều kiện (rẻ nhánh)
- Câu lệnh IF ... ELESE

```c#
if(điều kiện)
{
    // Khối lệnh nếu điều kiện đúng
}
else 
{
    // Khối lệnh nếu điều kiện sai
}
```

- Câu lệnh SWITCH ... CASE

```c#
switch(biến dò)
{
    case "giá trị 1":
        // Khối lệnh
        break;

    case "giá trị 2":
        // Khối lệnh
        break;

    case "giá trị 3":
        // Khối lệnh
        break;

    default:
        // Khối lệnh
}
```
